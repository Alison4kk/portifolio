# Meu Portfólio

Este é o projeto do meu portfólio pessoal.

Hospedado no GitHub Pages:

https://github.com/Alison4kk/portifolio

## Stack

- Vue 3
- Vite
- Tailwind CSS

## Comandos

```bash
npm install
npm run dev
```

Para gerar build de produção:

```bash
npm run build
```

Para publicar no GitHub Pages:

```bash
npm run deploy
```
